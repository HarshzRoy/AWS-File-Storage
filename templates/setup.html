Configure your S3 bucket for file storage access:

1. Configure CORS Policy

  1. Go to your AWS S3 Console
  2. Select your bucket ‚Üí Permissions tab
  3. Find Cross-origin resource sharing (CORS)
  4. Click Edit and paste the JSON below
CORS Configuration 
[
    {
        "AllowedHeaders": ["*"],
        "AllowedMethods": ["GET", "PUT", "POST", "DELETE", "HEAD"],
        "AllowedOrigins": ["https://aws-file-storage-harshs-projects-1d376aa0.vercel.app/"],
        "ExposeHeaders": ["ETag"],
        "MaxAgeSeconds": 3000
    }
]

2. Required IAM Permissions

Your AWS credentials need these S3 permissions:
s3:GetObject s3:PutObject s3:ListBucket s3:DeleteObject
